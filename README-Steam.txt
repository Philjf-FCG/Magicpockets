Magic Pockets Steam Build (WinUAE-only)

Contents:
- winuae64.exe
- magicpockets_steam.uae
- kickstart-1.3.rom
- magicpockets.adf
- magicpockets-plus4.adf
- magicpockets-skr.adf
- Launch-MagicPockets.bat

Recommended Steam launch setup:
1) Executable: winuae64.exe
2) Launch options: -f magicpockets_steam.uae
3) Working directory: game install root (same folder as winuae64.exe)

Alternative executable:
- Launch-MagicPockets.bat

Optional launch options for alternate disks:
- -s floppy0=magicpockets-plus4.adf
- -s floppy0=magicpockets-skr.adf
